4.1. Prise en compte des frais d’inscription ?

Comment faire payer par le labo une inscription à une conférence?

Réponse

Deux situations se présentent :

La conférence accepte des bons de commande

Il faut alors :

  • établir la demande d’achat (comme un achat classique)

  • faire établir une fiche fournisseur si la conférence n’est pas déjà connue des services comptables (cas le plus courant)

La conférence n’accepte pas les bons de commande (cas le plus courant)

Il faut alors avancer l’inscription et noter sur le bon de commande (dans la partie "renseignements complémentaires") le montant de l’inscription. Vous pouvez ensuite demander une avance de la mission pour prendre en compte la gène de l’avance.

Tous les documents sont sur l’ENT ou plus simplement à demander à Brahim HAMID.

4.7. Comment faire croire à ma machine qu’elle est à l’IRIT?

Je n’arrive pas à accéder aux publications des revues auxquelles l’IRIT est abonné?

Réponse

Pour être identifié à l’IRIT et visualiser les publies dans ScienceDirect par exemple, il est possible de faire un tunnel SSH et passer par le proxy :

ssh sash.irit.fr -l <login irit> -L12345:proxy.irit.fr:8001 -N

Que l’on peut mettre dans une boucle while pour éviter la déconnexion :

while true ; do ssh sash.irit.fr -l <login irit> -L12345:proxy.irit.fr:8001 -N ; done

Sous Windows, utiliser PuTTY pour faire la même chose (dans SSH > Tunnels)

Sous Linux comme Windows, vous pouvez passer par votre couple de clé publique/privée pour ne pas avoir à taper le mot de passe.

Une fois le tunnel établi, dans les proxys du navigateur web, indiquer 127.0.0.1:12345 comme proxy pour emprunter le tunnel.

il est possible d’utiliser un plugin comme proxy selector (module firefox) pour activer/désactiver le proxy sur un simple clic.
Voir aussi documentation VPN de l’IRIT
